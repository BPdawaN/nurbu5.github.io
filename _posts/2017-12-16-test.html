A pretty basic game I built in Unity by following these tutorials. The game might take a few seconds to load.

Instructions: Move using the arrow keys and shoot with the Alt key. If you lose you can start a new game with 'R' after the wave has ended.



In future iterations, I'll try to add a button to mute or lower the volume, and have the game resize with the window.
